How Does the Oil Changes For Life Program Work?

When you purchase a Pre-Owned vehicle from South Fort Chevrolet, you are automatically enrolled in the program. All you have to do to stay enrolled in the program is follow these guidelines:

  • The program is only applicable at SG Auto Locations
  • The program cannot be combined with any other offer
  • Regular service schedule must be followed to maintain Oil Change For Life program
  • If vehicle is serviced at another location, the program is void
  • This program cannot be sold or transferred
  • The program has no cash value
  • The program does not apply to As Traded vehicles
  • Synthetic oil is extra
  • Any special requests are extra
